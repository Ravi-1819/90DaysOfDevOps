ls- view files and folder in the current directory 
pwd- display the current working directory/folder
cd- change directory/folder
mkdir- create a new directory/folder
rmdir- remove an empty directory
touch- create a new empty file 
cp- copy files and directories 
mv- move or rename files and directories
rm- delete files 
cat- view file content
nano- edit files in the terminal
head- display the first lines of the file 
tail- display the last lines of a file
find- search for files and directories
grep- search for specific text in files 
ps- view runing processes
top- monitor live CPU and RAM usage 
kill- stop a runing process
systemctl- manage linux services
journalctl- view system logs
ping- check network connectivity
ip addr- display ip address information 
curl- fatch data from websites or APIs 
 Network Troubleshooting
 netstat- display network connection 
 ss- view active socket and network connection faster than "netstat"
 traceroute- show the path packets take to reach a destination server
