# Linux Architecture Notes

## Kernel
- Core part of Linux
- Manages CPU, memory, and hardware

## User Space
- User applications run here
- Examples: bash, python, chrome

## systemd
- First process after boot
- Manages services and startup

## Process Management
- Every running program is a process
- Each process has a PID

## Process States
- Running
- Sleeping
- Stopped
- Zombie

## Important Commands

### ps
Shows running processes

### top
Shows CPU and memory usage

### htop
Interactive process monitor

### systemctl
Manages services

### journalctl
Shows system logs
